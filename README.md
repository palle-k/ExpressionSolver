# ExpressionSolver 

A solver for simple mathematical expressions.

This project demonstrates the use of context free grammars
to solve expressions by traversing the syntax tree of the expression.

## Examples:

```
> -3^2^3
-6561.0
> tanh(pi/5+1/3*-e)
-0.270844926380908
> ((-e^5+log(sin(e+pi)+3))*(4^sin(-2)+4))
-631.652183876061
```
